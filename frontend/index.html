How to read the graph

Node colour = entity risk (green → red)
Node size = contribution (share of flow)
Edge colour = connection risk (same scale)
Edge width = money volume
➤
Arrow = direction money moves
Red halo ⚑ = screening hit

Interactions

hover node
Details + highlight. Tooltip with the entity’s risk, country and flags; its connections light up, the rest fade.
hover link
Money detail. Amount, direction and connection risk for that edge.
drag
Rearrange. Pull any node to untangle the layout.
click driver
Isolate. Shows only that typology’s sub-network; click again to reset.
